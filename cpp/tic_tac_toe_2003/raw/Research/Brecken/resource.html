 Compilers

  • DJGPP
    A free 32-bit DOS compiler. See also Brennan's DJGPP + Game Resources.
  • Watcom C/C++
    Version 11 supports DOS applications, and it's fast! See also Paul Hsieh's Home Page.|
  • Borland C/C++
    Borland sells TurboSuite, which has Turbo C++ 3.0. Turbo C++ 3.0 can be used for the DOS programs at this site.
  • Microsoft Visual C++
    For Windows and DirectX applications. They don't make a DOS compiler anymore.
